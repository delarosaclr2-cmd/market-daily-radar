# Reglas de análisis

## Reglas generales

1. No dar recomendaciones directas de compra o venta.
2. Separar oportunidades de estudio de movimientos puramente especulativos.
3. Ignorar penny stocks, microcaps y tokens de baja liquidez, salvo que sean relevantes para entender el sentimiento del mercado.
4. Priorizar activos líquidos y accesibles para un inversionista en México.
5. No presentar una caída fuerte como oportunidad solo porque cayó.
6. No presentar una subida fuerte como compra solo porque subió.
7. Si una causa no puede verificarse, decirlo claramente.

## Clasificación de movimientos

Clasificar cada activo en una de estas categorías:

- Movimiento con fundamento
- Movimiento especulativo
- Rebote técnico
- Movimiento por noticia
- Short squeeze posible
- Caída por deterioro fundamental
- No claro / requiere verificación

## Criterios para posible oportunidad

Un activo puede pasar a la sección de “posibles oportunidades para estudiar” si cumple al menos dos de estos puntos:

- Tiene volumen relevante.
- El movimiento está relacionado con una noticia verificable.
- Pertenece a un sector fuerte o relevante en el momento.
- Tiene una tesis clara de valor, momentum o recuperación.
- La relación riesgo/recompensa parece razonable.
- Existe una condición técnica clara para confirmar o invalidar la idea.

## Señales de advertencia

Marcar como “evitar por ahora” si hay:

- Baja liquidez.
- Movimiento tipo meme.
- Rumor no verificado.
- Riesgo regulatorio fuerte.
- Caída por fraude, investigación o deterioro estructural.
- Volatilidad excesiva sin información clara.
- Token o acción con capitalización demasiado baja.
